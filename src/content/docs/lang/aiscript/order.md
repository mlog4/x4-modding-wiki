For full content, see → **[Order definition](/lang/aiscript/order-definition/)**

## Why this redirect exists

The order concept is sometimes referenced as just "order" (short form) or "order definition" (full form). Both refer to the same aiscript construct — the aiscript file declaring how to execute a specific order (`<aiscript name="order.X">`).

For the game-side **Order** (the object representing an order on a ship's queue), see [Game API → Order](/game/behavior/order/).
