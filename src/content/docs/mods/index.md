User guides for X4 Foundations mods. Each page covers:

- **Short description** — what the mod does at a glance
- **Installation & configuration** — requirements, install options, in-game setup
- **Known conflicts** — mods that don't play well together (if any)
- **Deep dive** — long-form breakdown of the mod's features and game mechanics

We start with the mods we currently maintain (all X4 9.0-compatible, actively updated), and are gradually expanding to cover other popular community mods.

## Mods we maintain (v2.0.0, X4 9.0)

### Core dynamic-universe stack

- **[DeadAir Scripts (mlog4 fork)](/x4-modding-wiki/mods/deadair-scripts-fork/)** — Dynamic War, Xenon Evolution, Jobs Expeditions, Station Fill, and more. Standard variant with DA custom wares.
- **[DeadAir Scripts (No DA Wares fork)](/x4-modding-wiki/mods/deadair-scripts-no-wares-fork/)** — Same features, custom wares stripped. For VRO / X4-Reemergence / vanilla-economy players.
- **[DeadAir Economy Overhaul (mlog4 fork)](/x4-modding-wiki/mods/deadair-eco-fork/)** — Economy tuning, station enhancements, 3 custom wares. Standard companion to DA Scripts.
- **[DeadAir Economy Overhaul (No DA Wares fork)](/x4-modding-wiki/mods/deadair-eco-no-wares-fork/)** — Same tuning without custom wares. Clean pairing with VRO or vanilla economy.

### Modded-faction compatibility patches

- **[Apus Compat Patch](/x4-modding-wiki/mods/apus-compat/)** — Full DA integration for Apus Stellar Treaty faction.
- **[ETW Compat Patch](/x4-modding-wiki/mods/etw-compat/)** — Full DA integration for all 3 Eternal Twilight Expansion factions.

## Quick pick guide

**Playing vanilla X4 9.0 with the DA economy layer (custom wares included):**
→ [DA Scripts (standard)](/x4-modding-wiki/mods/deadair-scripts-fork/) + [DA Eco (standard)](/x4-modding-wiki/mods/deadair-eco-fork/)

**Playing with VRO 5.01 or X4-Reemergence (when available):**
→ [DA Scripts (No DA Wares)](/x4-modding-wiki/mods/deadair-scripts-no-wares-fork/) + [DA Eco (No DA Wares)](/x4-modding-wiki/mods/deadair-eco-no-wares-fork/)

**Also running Apus Stellar Treaty?**
→ Add [Apus Compat Patch](/x4-modding-wiki/mods/apus-compat/)

**Also running Eternal Twilight Expansion?**
→ Add [ETW Compat Patch](/x4-modding-wiki/mods/etw-compat/)

All combinations tested via 5.5-hour multi-mod soaks on X4 9.0. Zero errors in the mod namespace.
