The same viewer hub as [DA Blueprint Analysis → Progress Menus](../../configuration/blueprint-analysis/progress-menus/) — but reachable via **DA Information Menus** as well as via Blueprint Analysis's sub-menu.

## Why two entry points?

DA menu design consolidates diagnostic reports in `DA Information Menus` while also keeping feature-specific submenus accessible from each feature's configuration page. Progress Menus (for blueprint scan progress) is one of two reports that appears in both places (the other is [Fill Statistics](./fill-statistics/) for [DA Fill](../../configuration/fill/statistics/)).

Full documentation lives on the primary page: **[DA Blueprint Analysis → Progress Menus](../../configuration/blueprint-analysis/progress-menus/)** — go there for the 4 category sub-pages:
- [Station Modules](../../configuration/blueprint-analysis/progress-menus/station-modules/)
- [Ships](../../configuration/blueprint-analysis/progress-menus/ships/)
- [Equipment](../../configuration/blueprint-analysis/progress-menus/equipment/)
- [Misc](../../configuration/blueprint-analysis/progress-menus/misc/)
