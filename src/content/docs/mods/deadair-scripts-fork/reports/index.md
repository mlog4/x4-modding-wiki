DA ships several read-only info/report menus surfaced via **DA Menu → DA Information Menus**. Each report is a snapshot of live game state formatted for browsing.

> **📷 Screenshot needed:** DA Information Menus — root list of all reports.
> _File: `menu-info-root.jpg`_

## Reports

- **[Trade Stations Report](./trade-stations/)** — galaxy-wide list of trade stations with faction, sector, subordinate count, and storage capacity (mlog088-089, v3.0-beta1).
- **[Extension progress](./extension-progress/)** — observed clusters and Built lists for the Extension subsystem.
- **[Xenon Evolution status](./evolution-status/)** — current Xenon level, per-category equipment mod status, next-level ETA.
- **[Dynamic War stats](./war-stats/)** — event history, favor ledger, fatigue timers (requires `War_StatTracking = true`).
- **[Blueprint scan progress](./blueprint-progress/)** — per-ship-class blueprint fragment progress.

## Additional reports

Other reports may exist depending on installed subsystems — the report list is dynamic. If you have a submenu the docs don't cover here, please open a Nexus / GitHub issue with a screenshot.
