**Dockingbay** is a **single docking slot** — the specific position where a docking ship attaches. Each [Dockarea](/game/objects/dockarea/) contains multiple dockingbays.

## Properties

- `.parent` — the parent dockarea
- `.dockedship` — the ship currently docked here (if any)

## Common context

When a ship docks, it occupies a specific dockingbay. The dockingbay is the granular unit of dock occupancy.

## Related

- [Dockarea](/game/objects/dockarea/) — parent containing multiple dockingbays
- [Pier](/game/objects/pier/) — capital ship docking variant
- [Ship](/game/objects/ship/) — docks at a dockingbay
- [Station](/game/objects/station/) — has dockareas with dockingbays
